# WordCloud画像生成プログラム
このNotebookは文字を視覚的に表示するWordCloudを用いて、Gradioアプリとして実行できるようにしたものです。

## 使用環境
本プロジェクトは **Google Colab** 上での実行を想定しています。

## 使用したテキスト
このワードクラウド画像は、[青空文庫 Aozora Bunko](https://www.aozora.gr.jp/)内の「銀河鉄道の夜」テキストを用いてWordCloudを生成しています。


<img src="https://github.com/honomax/gradio-wordcloud-app/blob/main/sample.webp?raw=true" width="500" alt="アプリケーションのスクリーンショット">

## 結果・出力
Notebookを実行すると、Gradioアプリが起動。

UI上で出力されたWordCloud画像の保存が可能です。
